gitdig

CLI application archetype

'gitdig' is a simple git tool for documenting and dicovering multi-repo projects.

Get Started

Collect Repository info

Collect the most important information about the git repositories into a json or yaml file, for further processing..

Collect Package info

Collect package info via downloading package.json (pom.xml, or other package file)

List and Check respos

List and Check repositories via predefined rules, and print problems to the console.
